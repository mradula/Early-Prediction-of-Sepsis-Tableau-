# Early-Prediction-of-Sepsis-Tableau
An interactive healthcare analytics project focused on exploring biomarker patterns across sepsis and non-sepsis patients using Tableau. This project demonstrates how visual analytics can simplify complex ICU datasets and generate insights that support early sepsis detection and clinical decision-making.

# Project Overview
Sepsis is a life-threatening condition that requires early identification for effective treatment. In this project, I analyzed biomarker data from ICU patients and developed interactive Tableau dashboards to compare sepsis and non-sepsis cohorts.
The goal was to transform raw clinical data into intuitive, decision-friendly visuals that highlight patterns in key biomarkers such as lactate, WBC count, and other clinical indicators associated with sepsis progression.
